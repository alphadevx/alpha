Class AlphaHighlightProviderGeshi

Wraps Geshi in the standard API defined in AlphaHighlightProviderInterface, for use in the Alpha CMS module. Note that Geshi is not included in Alpha, so you will need to download it from here:

http://qbnz.com/highlighter/

Once downloaded, place in the [application]/lib/geshi directory which is where this class will attempt to load it from.
