Class: Image

Class Image

Class Overview
Class Overview | Method Summary | Methods Detail

A widget that can generate an image which is scaled to the screen resolution of the user, and can be made secure to prevent hot-linking from remote sites. Note that by default, a jpg file will be returned (the source file can be jpg, png, or gif).
